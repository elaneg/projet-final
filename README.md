# projet-final : description rapide du projet

#Projet de fin de premier semestre en M1 TAL

#Il consiste à créer un script qui va parcourir 3 corpus de 50 URLs (150 au total) de 3 langues afin d'analyser les différences sémantiques d'un mot choisi. Nous avons choisi le mot "culture". 
#Les différentes données traitées sont présentées dans un tableau et le tout est mis en ligne sur ce site.
#Un nuage de mots montrant les mots revenant le plus souvent autour du mot "culture" a été généré pour chaque langue via un deuxième script, et les nuages sont postés sur le site.
